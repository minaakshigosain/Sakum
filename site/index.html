Self-Learning Engine

      
Survivability metrics collected after each compile/run. Failures go into a mistake ledger; the agent rewrites code paths to avoid repeating them. Live from memory.md & learn.md. The formal spec is documented in docs/LATEX_SYNTAX.md and docs/SYMBOL_MAP.md; the verified spec corpus is embedded as raw assembly in docs/asm/spec_*.s (mac build prints FNV1a(spec)=0xC46A785B).
